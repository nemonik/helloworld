# GoLang Helloworld [![Build Status](https://travis-ci.org/nemonik/helloworld.svg?branch=master)](https://travis-ci.org/nemonik/helloworld)

A simple helloworld GoLang app.

### Prerequisites

GoLang must be installed and configured.

### Testing

Typcially

```
make test
```

Alternatively

```
make sonar
```

### Running

```
make run
```
